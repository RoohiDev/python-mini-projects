# Age Calculator

A simple program that calculates your age based on birth year.

## How it works

1. Gets current year from your system using datetime module
2. Asks for your birth year
3. Subtracts birth year from current year
4. Shows your age

## Run

```bash
python main.py
```

## Example

Enter your birth date year: 2000

You are 26 years old.

## Dependencies

None (uses only Python built-in modules)
